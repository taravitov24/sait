Глосарій термінів

120+ ключових понять мережевих технологій

OSI — Open Systems Interconnection. Еталонна 7-рівнева модель.
TCP — Transmission Control Protocol. Надійний протокол транспортного рівня.
IP — Internet Protocol. Протокол мережевого рівня.
MAC-адреса — Фізична адреса мережевого адаптера (Layer 2).
Router — Пристрій, що працює на 3-му рівні моделі OSI.
Switch — Пристрій 2-го рівня, що комутує трафік за MAC-адресами.
DHCP — Dynamic Host Configuration Protocol.
DNS — Domain Name System. Система доменних імен.
QoS — Quality of Service. Гарантія якості обслуговування.
VLAN — Virtual Local Area Network.
